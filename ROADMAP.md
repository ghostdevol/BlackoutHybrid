# BlackoutHybrid Roadmap
Public Overview — Non‑Classified

This roadmap outlines the planned public-facing features for the
BlackoutHybrid wrapper. Internal engine features are not listed here.

---

# 1. Near-Term Goals

- Stabilize Android wrapper
- Improve build pipeline reliability
- Add modular UI panels (Explorer, Wallet, NFT)
- Harden bridge layer teardown routines

---

# 2. Mid-Term Goals

- Expand wrapper to support additional GhostTech modules
- Introduce offline-first UX improvements
- Add optional local-only caching (non-sensitive)
- Improve error reporting (non-sensitive)

---

# 3. Long-Term Goals

- Full integration with GhostTech SUITE Elite
- Unified dashboard architecture
- Multi-environment support (Android, Desktop, WebView)
- Optional plugin system for non-sensitive extensions

---

# 4. Out of Scope

The following will never be included in this repository:
- Blackout Wallet Engine
- Cryptographic routines
- Key lifecycle management
- Mediation controllers
- Classified execution boundaries

These remain private and proprietary.

---

End of public roadmap
